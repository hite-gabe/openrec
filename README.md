 OpenRec is an open-source screen recorder for Debian-based Linux desktops. 

 | Distribution / OS | Desktop Environment | Status | Notes |
| :--- | :--- | :--- | :--- |
| **Linux Mint** (21.x / 22.x) | Cinnamon, XFCE, MATE | **Fully Compatible** ✅ | Native environment; seamless menu integration. |
| **Ubuntu** (22.04 / 24.04 / 26.04) | GNOME, XFCE, KDE | **Fully Compatible** ✅ | Requires X11 session (switch from Wayland if needed). |
| **Debian** (11 / 12) | Any | **Fully Compatible** ✅ | Architecture matches natively; install via apt. |
| **Pop!_OS** | COSMIC / GNOME | **Fully Compatible** ✅ | Uses apt and standard Debian packages. |
| **Arch Linux / Fedora** | Any | *Unsupported via .deb* ⚠️ | Source code runs via Python, but package requires conversion. |
| **Windows / macOS** | N/A | *Not Supported* ❌ | Relies on Linux-specific X11 and FFmpeg tools. |
